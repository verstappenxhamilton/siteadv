# Triagem Jurídica

Aplicação de chat para triagem jurídica. Utiliza Next.js, React, Tailwind e suporte a múltiplos provedores de IA.

## Desenvolvimento

```bash
npm install
npm run dev
```

Acesse `http://localhost:3000`.

## Variáveis de ambiente

Copie `.env.example` para `.env` e preencha as chaves dos provedores de IA.

## Scripts

- `npm run dev` – modo desenvolvimento
- `npm run build` – build de produção
- `npm start` – inicia servidor de produção
