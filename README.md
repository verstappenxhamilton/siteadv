# siteadv

## Scripts

- `npm install` – instala as dependências do servidor e do bundler do frontend.
- `npm run build` – compila os arquivos React presentes em `src/` para `public/assets/` usando o esbuild. Execute este comando sempre que fizer alterações no código fonte do site.
- `npm start` – inicia o servidor Express servindo os arquivos gerados em `public/`.
