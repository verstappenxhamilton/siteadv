# SiteAdv Chat Jurídico

Aplicação Next.js para triagem jurídica com chat.

## Rodando

```bash
npm install
npm run dev
```

## Rotas

- `/` – seleção de área e início da triagem
- `/chat` – interface de chat
- `/settings` – configurações (salvas no browser)

## Variáveis de ambiente

Veja `.env.example`.
